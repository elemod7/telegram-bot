🤖 Telegram-бот для сотрудников продаж

🚀 Возможности:

📂 Работа с файлами и документами
Отправка PDF-файлов по категориям и кнопкам меню (pdfs/, docs/)
Проверка наличия файла перед отправкой
Отображение уведомления "Файл отправляется..." и его автоматическое удаление
Поддержка вложенных меню (через callback'и)

🔐 Контроль доступа
Проверка доступа при выполнении защищённых команд
Хранение списка разрешённых пользователей в allowed_users.json
Отдельная команда для администратора /list_allowed_users
Возможность удалить пользователя из доступа через /remove_access
Отказ в выполнении команд для неавторизованных пользователей

👤 Запрос доступа
Кнопка "Запросить доступ" для новых пользователей
Сохранение заявок в access_requests.json
Уведомление админа о новой заявке
Возможность админа нажать "Предоставить доступ" прямо из чата
Сохранение имени и ID пользователя при предоставлении доступа

💬 Команды и взаимодействие
/start — Приветствие, запуск главного меню
/admin — Доступ к админ-панели (только для админов)
/list_allowed_users — список всех пользователей с доступом
/remove_access — удалить пользователя по ID
Кнопка "Вернуться в меню" — доступна во всех разделах
Кнопка "Запросить доступ" — если пользователь без доступа

📊 Логирование
Логирование действий пользователей в SQLite (bot_usage.db)
Лог-файл bot.log с событиями и ошибками
Логирование каждого открытия меню, команды и запроса доступа

⚙️ Архитектура и админка
Разделение на модули: handlers/, database.py, shared.py, access_management.py
Легко добавлять новые команды и меню
Настройки доступа вынесены в shared.py и .env
Используется aiogram 3.x и asyncio
Возможность редактировать контент из JSON-файлов

🧰 Dev-фичи
.gitignore с исключениями для логов, баз и venv
reset_and_push.sh — скрипт для полной перезаливки GitHub
requirements.txt — зависимости проекта
Возможность запуска через systemd на сервере
Интеграция с логикой FSM при необходимости

⚙️ Установка
Клонируйте репозиторий:
git clone https://github.com/elemod7/telegram-bot.git
cd telegram-bot

Создайте виртуальное окружение:
python3 -m venv venv
source venv/bin/activate

Установите зависимости:
pip install -r requirements.txt

Создайте .env файл на основе example.env и вставьте ваш токен бота:
BOT_TOKEN=ваш_токен_здесь
ADMIN_ID=123456789
